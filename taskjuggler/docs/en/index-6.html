6. Other useful commands and environments

now 2002-02-17

This sets the current date to 2002-02-17. This is the day/week that will be higlighted in the report later. This helps you printing/creating reports for the future.

rate 310.0

currency "EUR"

Sets the global monetary currency to Euro. It is used for printing out cost and revenue values in the correct currency.

account dev "Development" cost {}

This creates an account dev as a cost account. There is also a account type revenue, which is used to cover all kinds of income (see reference: account).
