1. Introduction

1.1 What is TaskJuggler

TaskJuggler is a project management tool. It helps you to organize and plan projects in your company. It schedules the whole project and tries to optimize the resources needed for the project.

The output can either be a report showing the several tasks, the resource usage or the account information. Multiple reports can be created as html files which enables an easy display via a WEB browser.
